前端：HTML+CSS📠
  • 基础知识
    • 网页：网站是网页的集合，网页是构成网站的基本元素，因其由html后缀结尾，俗称html文件
    • HTML：超文本标记语言，是标记语言，不是编程语言
    • 浏览器内核：渲染引擎
      • google：Blink
      • apple：Webkit
    • Web标准：W3C（万维网联盟）
      • 最佳体验方案：结构、样式、行为
      • 结构写到HTML中，表现写在CSS中，行为写在Javascript中
    • Web标准的构成：
      • 结构：Structure
        • 对网页元素进行整理和分类，现阶段主要是html
      • 表现：Presentation
        • 用于设置网页元素的板式、颜色、大小等外观样式，主要是css
      • 行为：Behavior
        • 网页模型的定义及交互的编写，javascript

    • 常用标签
      • 标题标签
        • <h1></h6>：一级标题-六级标题
          • 标签语义：作为标题使用，依据重要性递减
          • 特点：标题文字会变粗，字号会依次变大，每个标题独占一行
      • 段落标签（paragraph）
        • <p>我是一个段落标签</p>
          • 会根据浏览器自适应调整
      • 换行标签
        • <br/>：break，单标签
        • 无行距，和段落标签的区别
      • 文本格式化标签
        • 设置粗体、斜体、下划线等
          • 加粗<strong></strong>或<b></b>
          • 倾斜<em></em>或<i></i>
          • 删除线<del></del>或<s></s>
          • 下划线<ins></ins>或<u></u>
      • div标签与span标签----division
        • 没有语义，盒子装内容
        • div分割、分区；span跨度跨距
          • div标签用来布局，一行只能放一个div，大盒子
          • span，小盒子一行可以放多个span
      • 图像标签（单标签）
        • <img src="图像URL"/>
        • 图像属性：
          • src(source)是<img>标签的必须属性，用于指定图像的文件路径与名称
          • title属性：提示文本，鼠标放到图像上的提示文字
          • width：设置图像宽度，同理height高度、border图像边框粗细
            • width="500"，高度和宽度一般只修改一个，会自适应调整
        • 相对路径
          • 同级路径<img src="文件名"/>
          • 下一级路径<img src="文件夹名/文件名"/>
          • 上一级路径文件名<img src="../文件名"/>
        • 绝对路径
          • <img src="C:\Users\GuoJia\Desktop\文件名"/>
          • 网络绝对地址：<img src="www.baidu.com/文件名"/>
        • 超链接标签<a>,anchor
          • <a href="跳转目标" target="目标窗口弹出方式">文本或图像</a>
            • href是必须属性，指定链接地址
            • target指定打开方式，_self为默认值，_blank为在新窗口中打开
          • 链接分类
            • 外部链接：以http://开始
            • 内部链接：<a href="xxxx.html">链接文本</a>
            • 空链接<a href="#">链接文本</a>
            • 下载链接：href里面的链接是一个文件或压缩包，会下载这个文件
            • 网页元素链接：
              • 把图片当链接媒介，<a href="http://www.baidu.com"><img src="img.jpg"/></a>
          • 锚点链接：网页分段，快速定位到网页某个位置
            • 在链接文本的href属性中，设置属性值为#名字的形式，如<a href="#two">第二集</a>
            • 在目标位置标签，添加id属性 = 刚才的名字，如<h3 id="two">第二集介绍</h3>

  • HTML标签下
    • 表格标签
      • 基本元素：table、tr、td
        • table定义表格标签；tr定义行，嵌套在table中；td定义单元格，嵌套在tr中
        • 表头单元格标签：th----table head会加粗、并居中显示
        • thead与tbody
        • 合并单元格
          • 方式
            • 跨行合并：rowspan="合并单元格的个数"，最上侧单元格作为目标单元格
            • 跨列合并：colspan="合并单元格的个数"，最左侧单元格作为目标单元格
            • <td colspan="2"></td>，合并完成后删除多余单元格
    • 列表标签
      • 列表：整齐、有序作为布局会自由方便
      • 无序列表、有序列表、自定义列表
        • 无序列表（重点）
          • <ul>定义无序列表，列表项用<li>定义，无顺序之分
          • <ul>中只能嵌套<li>，不能放其他标签，<li>中可以放其他标签和元素
        • 有序列表：各个列表项按照一定的顺序排列定义
          • <ol>定义有序列表，列表项用<li>定义
        • 自定义列表：上方标题，下方围绕分项目（网页底部）
          • <dl>定义有序列表，<dt>定义父项目名称（标题）,<dd>定义子项目名称
    • 表单标签
      • 用户注册页面，收集用户信息
      • 组成成分：表单域、表单控件（表单元素）、提示信息
        • 表单域：包含表单元素的区域，实现用户信息的收集与传递
          • <form>标签用于定义表单域，把范围内的表单元素提交给服务器
          • 表单域属性：action、method、name
            • action：属性值url地址，用于指定接收并处理表单数据的服务器程序的url地址
            • method：属性值get/post，用于设置表单的提交方式，其取值为get或post
            • name：属性值名称，用于指定表单的名称，以区分同一个页面中的多个表单域
        • 表单控件：允许用户在表单中输入或选择的内容控件
          • input输入表单元素
            • 收集信息
            • <input type="属性值"/>,type用于指定不同类型的表单元素控件，文本，复选框，按钮等
              • 例如<input type="password">----输入密码框、<input type="radio">男----性别的单选按钮
              • button按钮
                • <input type="button" value="获取短信验证码">
              • checkbox定义复选框
              • file定义输入字段和文件上传
                • <input type="file" >
              • hidden定义隐藏的输入字段、image定义图像形式的提交按钮
              • password定义密码字段，字符被掩码、radio定义单选按钮（多选一）、reset定义重置按钮，清除表单所有数据
              • submit定义提交按钮，把表单数据发送到服务器
                • <input type="submit" value="立即注册"> ----实现一个"立即注册"的按钮
              • text定义单行的输入字段，用户可输入文本，默认宽度20字符
              • name属性：给表单起名字。例如实现性别的多选一
                • 男<input type="radio" name="sex"> 女<input type="radio" name="sex">
              • checked：规定此input元素首次加载时应当被选中（打开页面，默认选择，例如用户协议）
              • label标签：为input元素定义标注
                • 扩大复选框的选择范围，例如选择男女头像即可选择性别，不用再去点小圈选项
          • select下拉表单元素
            • 父标签select,子标签option
              • <select>
                • <option>四川</option>
                • <option>江苏</option>
                • <option>北京</option>
              • </select>
              • 在select中定义select="selected"时，当前项即为默认项
          • textarea文本域表单元素：网站介绍，留言板等多行输入时
  • CSS层叠样式第一天----Cascading Style Sheets
    • 简介
      • 设置HTML中的文本内容（字体、大小、对齐方式）、图片形式（宽高、边框样式、边距）、版面布局、外观样式
      • 语法规范：选择器+声明样式（属性+值）
      • CSS写在style中，位于head中
      • 代码风格（风格）：
        • 紧凑型格式、展开型
        • 冒号后保留一个空格
        • 在选择器和大括号之间保留一个空格
    • CSS基础选择器
      • 选择器的作用：根据不同的需求选择出不同的标签
        • 基础选择器：由单个选择器组成：包括标签选择器、类选择器、id选择器、通配符选择器
          • 标签选择器：以HTML标签名称作为选择器，为某一标签指定CSS样式
          • 类选择器：选择一个或某几个标签
            • 案例：
            • 多类名：给一个标签指定多给类名
              • 在标签class属性中写多个类名
              • 多个类名中间必须用空格分开
              • 例如既更改字体颜色，又更改字体大小和样式，class调用两个内
        • 复合选择器
          • id选择器：id属性来设置Id选择器,id选择器以#来定义
            • ID选择器和类选择器的区别在于，ID选择器只能调用一次，其他项不能使用该ID
            • 类选择器类似姓名，ID选择器类似身份证号码
          • 通配符选择器：选取页面中所有的元素（标签）
    • CSS字体属性：定义网页中的文字属性：字体样式、大小等
      • font-family：字体，如果有多个字体用逗号分隔
        • 浏览器会先识别第一个字体，如果用户电脑未安装改字体，则识别第二个以此类推
      • font-size:50px设置字体大小，给页面所有字体指定统一大小
      • font-weight:设置字体粗细
        • normal默认、blod加粗、bolder特粗、lighter细体、number（数字设置粗细）
        • font-weight:700等价于font-weight:bold----font-weight:400等价于font-weight:normal
      • font-style：文字样式（斜体与否）
        • font-style：normal正常、font-style：italic斜体
    • CSS文本属性：外观、下划线、装饰性文本、行间距、缩进等
      • 颜色：十六进制#ff0000；RGB代码rgb(255,255,0)
      • 对齐文本：text-align文本的水平对齐，居中、左、右
      • 装饰文本：text-decoration，添加下划线、删除线、上划线等
        • text-decoration：none----默认（删除超链接的下划线）
        • text-decoration：underline----下划线
        • text-decoration：overline----上划线
        • text-decoration：line-through----删除线
      • 文本缩进（首行）：text-indent
        • text-indent：20px;
        • text-indent：2em;------两个文字的距离
      • 行间距:line-height（上间距+文本高度+下间距）
        • line-height：20px
    • CSS引入方式
      • 行内样式表（行内式）
        • 在标签内部写style，例如<p style="color:pink">段落</p>
        • 适用于简单修改单个标签样式
      • 内部样式表（嵌入式）
        • 写在style标签中，控制当前HTML页面
      • 外部样式表（链接式）----最常用
        • CSS与HTML分离
        • 新建后缀为CSS的样式文件，把所有CSS代码放在此文件中
        • 使用link链接HTML文件与CSS文件
          • 语法规范，在HTML中head里写：<link rel="stylesheet" href="CSS文件路径">
    • 调试工具
      • Elements：元素（左边HTML，右边CSS）
  • CSS层叠样式第二天
    • emmet语法：提高开发效率
      • 快速生成HTML结构语法
        • 标签名+tab
        • 标签名*N，快速生成多个标签
        • 生成带有类名或者id名字的，直接写.类名或#名
      • 快速生成CSS样式语法
        • 简写
        • 格式化代码：右键、格式化文档
    • CSS复合选择器：基础选择器的叠加
      • 后代选择器、子选择器、并集选择器、伪类选择器
      • 后代选择器：（包含选择器）----选择父元素里面的子元素
        • style中写明标签如ol li{ }----元素1（父） 元素2（子） { 样式声明 }
      • 子选择器：选择最近一级的子元素----元素1（父）> 元素2（子） { 样式声明 }
      • 并集选择器：选择多组选择器----元素1 ，元素2 { 样式声明 }
      • 伪类选择器：用冒号表示，链接伪类
      • focus伪类选择器：用于选取获得焦点（光标）的表单元素
    • CSS元素显示模式
      • 元素（标签）以什么样的方式进行显示
        • 例如div独占一行，而一行可以放多个span
          • 块元素（独占一行）、行内元素
      • 元素显示模式转换：一个模式的元素需要另外一种模式的特性
        • 增加链接<a>的触发范围：display:block
        • 转换为行内元素：display:inline
        • 转换为行内块元素：display:inline-block(一行放多个div等等)
      • 元素显示模式分类
        • 块元素：<h1~h6>、<p>、<div>、<ul>、<ol>、<li>
          • 独占一行
          • 高度、宽度、外边距、内边距都可以控制
          • 默认宽度是容器(父级)宽度的100%
          • 是一个容器及盒子，里面可以放行内元素或块级元素
          • 文字标签内不放块元素
        • 行内(内联)元素：<span>、<strong>、<b>、<a>、<i>、<u>、<s>、<del>、<em>
          • 相邻行内元素在一行，一行可以显示多个
          • 宽高设置无效
          • 默认宽度为本身
          • 行内元素内部只能容纳行内元素
        • 行内块元素：<img/>、<input>、<td>同时具有块元素和行内元素的特点
          • 和相邻行内元素（行内块）在一行上，但他们之间会有空隙，一行可以显示多个（行内元素的特点）
          • 默认宽度为本身（行内元素的特点）
          • 高度、宽度、外边距、内边距都可以控制（块级元素的特点）
    • CSS背景：背景颜色、背景图片、背景平铺、背景图片位置、背景图像固定
      • 背景颜色：background-color：属性值（十六进制、RGB）
      • 背景图片：background-image：url----图片、logo、装饰性icon，易于控制位置
      • 背景平铺：background-repeat：repeat、no-repeat、repeat-x、repeat-y,默认都是平铺的
      • 图片位置：background-position：x y（可以使用方位名词上下左右、精确单位px，%等）
        • background-position：center top----顶部居中（方位词顺序不限）
        • background-position：20px 50px，左侧20px，上侧50px
      • 背景图像固定：背景是否随着页面滚动：background-attachment（scroll、fixed）
        • background-attachment：scroll----滚动
        • background-attachment：fixed----固定
      • 背景色半透明：background：rgba(x,x,x,a)
        • a:0~1，即Alpha的缩写
    • CSS三大特性：层叠性、继承性、优先级
      • 层叠性：相同选择器设置了不同样式，为了解决样式冲突问题
        • 存在冲突则就近原则，未冲突的部分依旧执行
      • 继承性：子标签继承父标签样式
        • 例如div中的p标签，会继承div的属性
      • 优先级：给同一元素指定多个选择器，优先级几何？
        • 选择器相同，则就近原则
        • 选择器不同，!important>行内样式style>ID选择器>类选择器、伪类选择器>元素选择器>继承或*
